# PREX FAQ

## Is this the PREX source-code repository?

No. This repository is only a product showcase. It contains screenshots, product descriptions, feature notes, and roadmap information.

## Where can I try PREX?

Open [https://prex.best](https://prex.best).

## What markets does PREX focus on?

PREX currently focuses on prediction-market workflows, especially Polymarket, and is also exploring on-chain opportunity detection through the Meme Scanner.

## What is the Meme Scanner?

The Meme Scanner is a token discovery and risk-screening interface. It ranks small-cap tokens using trading activity, liquidity, turnover, momentum, social heat, holder structure, and risk-concentration signals.

## Does PREX custody user funds?

PREX is designed around user-controlled wallets. Product details may evolve, but the goal is to reduce trading friction without forcing users into a fully custodial flow.

## Does this repository include trading strategies?

No implementation code is included here. Strategy ideas may be described at a high level, but source code and private trading logic are not part of this showcase repository.
