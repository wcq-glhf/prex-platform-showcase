# PREX Roadmap

This roadmap is product-facing and may change as the platform evolves.

## Near Term

- Improve Polymarket trading flow and reduce wallet friction.
- Expand prediction-market strategy testing.
- Improve Meme Scanner ranking quality and risk labeling.
- Add more token detail signals.
- Improve mobile layout for dense trading and scanner views.
- Add more public product screenshots and videos.

## Strategy Tooling

- More robust strategy dashboards.
- Better paper-trading analytics.
- Cleaner comparison between simulated PnL and real execution.
- More market categories beyond crypto/weather.
- Better handling of fees, slippage, order cancellation, and market settlement.

## Meme Scanner

- Add more chains and token sources.
- Add watchlists.
- Add alerting for score changes and liquidity shifts.
- Add richer holder and cluster analysis.
- Add social mention history.
- Add stronger risk labels for suspicious token structures.

## Growth And Community

- Improve invite flow.
- Add clearer referral reward status.
- Add public-facing product docs.
- Add case studies and strategy writeups.

## Long Term

- Multi-platform prediction-market aggregation.
- Copy-trading and strategy marketplace workflows.
- AI-assisted market research.
- Unified portfolio and risk view across prediction markets and on-chain opportunities.
